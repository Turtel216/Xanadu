# Xanadu

#### A simple interpreted programming language with syntax based on Rush song lyrics and titles

## Syntax

### Variables

The yyz keyword is used for declaring variables.

        yyz name = "Xanadu";

### Functions

Functions are declared using the **subdivision** keyword 

        subdivision foo() { }
